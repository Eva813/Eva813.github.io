Vue.js 3 - 起手式

2021-05-16

今天要介紹的是 Vue.js3 的起手式，主要參考官方文件來學習

載入方式

<script src="https://unpkg.com/vue@next"></script>

引入之後，在index.html檔案

 <div id="app">
    <h1>{{message}}</h1>
  </div>
//JS

Vue.createApp({

  data(){
     return {
      message: 'I am Eva!'
    }
  }
}).mount('#app')
